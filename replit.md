Statement of Account (SOA) Generator - AMOY Mannequins

Overview

A professional web-based Statement of Account generator for AMOY Mannequins (Abdulrahman Mohamed Othman Yousef EST.) that allows creating supplier statements and exporting them as PDF documents.

Project Structure

/

├── script.js            # Application logic, form handling, PDF generation
├── attached_assets/
│   ├── logo.png         # AMOY Mannequins logo
│   └── stamp.png        # Company stamp for signatures
└── replit.md            # Project documentation

Features

Modern UI with Inter font and clean styling
Fixed company header with logo and address
Dynamic invoice line item entry with auto-calculated balance
Real-time table updates with add/delete functionality
Automatic calculation of totals (invoiced, paid, outstanding)
Professional preview layout with company branding
PDF generation with logo, stamp, and complete formatting
SAR (Saudi Riyal) as default currency
Technologies

HTML5, CSS3, Vanilla JavaScript
Google Fonts (Inter)
jsPDF library for PDF generation
jsPDF-AutoTable plugin for formatted tables
Running the Application

The app is served using Python's built-in HTTP server on port 5000.
